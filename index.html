Introduction

Search for tools designed to support sustainable and healthy communities. Two methods are provided for finding these tools.

Tool Finder is an interactive wizard that leads users to relevant tools after answering questions. Tool Finder is a guided exploration that allows users to find tools by answering questions. This method of finding tools might suggest tools that the user may not have considered searching for.

Keyword Search allows users to search for text in titles and descriptions of tools. This textual search engine searches titles and descriptions for text a user enters. Any tools that match the text entered are shown. This method allows users to search in a specific way by entering specific text.

Tool Finder

What is It?

The Tool Finder is an interactive online search engine. It provides an intuitive way for users to search for relevant tools designed to support sustainable and healthy communities.

Why do I need it?

The READ database contains hundreds of tools designed for practitioners and decision makers. The Tool Finder guides users in locating directed tools based on their unique objectives and interests.

How do I use it?

Using a guided search engine, users narrow their search for tools by identifying their role within the community (Who are you?), specifying their objectives (What do you want to accomplish?), and selecting the relevant concepts that correspond to directed tools (Where do you want to direct your focus?).

This Tool Finder is not intended to prescribe solutions, but instead is meant to deliver a range of tools for decision makers.

Keyword Search

The Environmental Protection Agency’s Sustainable and Healthy Communities Research Program (SHCRP) has identified and evaluated an inventory of decision support tools designed to guide non-experts and professionals who seek solutions to community level development needs that will sustain human wellbeing and ecological functions.

This web-based inventory provides a searchable resource for building owners, developers, municipal planners, architects, engineers, and other interested community stakeholders. The tools in the inventory provide data, model scenarios, and narratives that guide the user through the decision-making process and promote sustainability at the community level.

The tools in the inventory occupy the following four decision sectors:

Buildings and Related Infrastructure
Decisions around buildings, like green building, or infrastructure, like stormwater management designs or extension of sewer
Land Use
Decisions around zoning, development, parks, etc
Transportation
Decisions around roads, transit, bike/pedestrian infrastructure, etc
Waste Disposal
Decisions around waste management, and options to avoid landfilling waste, like recycling, composting or programs to divert wasted food or “ugly produce”

Tools in the Inventory

The inventory of tools is kept in the Registry of EPA Applications, models, and Data-warehouses(READ). READ is a comprehensive registry of informational resources for the EPA. General information about READ is here. The inventory is accessed using READ Web Services.

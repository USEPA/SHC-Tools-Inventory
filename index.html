The Tool Inventory

Supported by EPA's Sustainable and Healthy Communities Research Program (SHCRP) this inventory provide data, model scenarios, and narratives that guide the user through decision-making processes in ways that promote sustainability at the community level. The current inventory contains 110 tools, but the content pool will expand and contract as new tools are added and old tools are phased out. If any links to tools appear broken or missing, please let us know.

The tools in the inventory occupy the following four decision sectors:

Buildings and Related Infrastructure
Decisions around buildings, like green building, or infrastructure, like stormwater management designs or extension of sewer.
Land Use
Decisions around zoning, development, parks, etc.
Transportation
Decisions around roads, transit, bike/pedestrian infrastructure, etc.
Waste Disposal
Decisions around waste management, and options to avoid landfilling waste, like recycling, composting or programs to divert wasted food or “ugly produce.”

The inventory of tools is kept in the Registry of EPA Applications, models, and Data-warehouses(READ). READ is a comprehensive registry of informational resources for the EPA. General information about READ is here. The inventory is accessed using READ Web Services.

Guided Search

Go to Guided Search.

What is it?

The Guided Search is an interactive online search engine. It provides an intuitive way for users to search for relevant tools designed to support sustainable and healthy communities.

Why do I need it?

The tool inventory contains tools designed for practitioners and decision makers. The Guided Search guides users in locating directed tools based on their unique objectives and interests.

How do I use it?

Using a guided search engine, users narrow their search for tools by identifying their role within the community (Who are you?), specifying their objectives (What do you want to accomplish?), and selecting the relevant concepts that correspond to directed tools (Where do you want to direct your focus?).

How does it work?

The Guided Search is build on top of the Decision Path Library (DPL). The DPL is a database that organizes elements of environmental management decisions. The database categorizes and relates information on who is making a decision, what objectives are being accomplished, what management options are being used to accomplish the objectives, and how progress can be measured using indicators and available data.

READ tools are tagged with concepts taken from the DPL. We use your selected roles and objectives to determine related concepts. This allows us to provide a list of tools relevent to accomplishing your objectives.

This Guided Search is not intended to prescribe solutions, but instead is meant to deliver a range of tools for decision makers.

Keyword Search

Go to Keyword Search.

Keyword Search is used to search for text in a tool's title, acronym, keywords, or description. Advanced search lets users search for text contained in titles, acronyms, descriptions, keywords, model-inputs, or output-variables. Advanced search lets users require that tools

  • occupy a selected decision-sector,
  • operate within a selected environment,
  • cost a specified amount, and
  • pertain to a specified spatial extent.

Keyword Search has an autocomplete feature that helps users explore what text is present.

Keyword Search works by searching for matches in the acronym, name, description, or keywords fields in the READ tool inventory.

Browse Tools

Additional Resources

These inventories may contain resources which are outside the scope of this tool inventory:

  • Green Infrastructure Wizard (GIWIZ) is a repository of EPA-sourced Green Infrastructure tools and resources designed to support and promote sustainable water management and community planning decisions.
  • The Superfund Community Involvement Toolkit (CI Toolkit) provides Superfund Regional site teams, community involvement staff, and others with a practical easy-to-use aid for designing and enhancing community involvement activities.
  • SHIFT Exit contains business sustainability resources curated by the Sustainability Initiative at MIT Sloan.
  • The Ecosystem-Based Management (EBM) Tools Network Exit is a premier source of information and training for planning and management tools for coastal-marine ecosystem based management (EBM), marine protected area (MPA) and marine spatial planning (MSP) and integrated land-sea planning.
